Setup Steps

  1. Create Firebase Project
    • Go to Firebase Console
    • Create a new project
    • Enable Firestore Database
    • Enable Email/Password Authentication
  2. Get Firebase Config
    • In Firebase Console, go to Project Settings
    • Scroll down to "Your apps"
    • Click on Web app icon (</>)
    • Copy the firebaseConfig object
  3. Update Configuration
    • Replace the Firebase config in index.html
    • Replace the Firebase config in login.html
    • Replace the Firebase config in dashboard.html
  4. Set Firestore Rules
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read: true;
        }
        match /{document=**} {
          allow write: if request.auth != null;
        }
      }
    }
  5. Create Admin User
    • Go to Firebase Console > Authentication
    • Click "Add user"
    • Enter your email and password

Collections Structure

Firebase will automatically create these collections when you add data:

  • about - Your about information
  • skills - Your skills and technologies
  • experience - Work experience
  • projects - Your projects
  • blogs - Blog posts
